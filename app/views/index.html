This prototype lets you access descriptions of documents that will be included in the trust framework. Some documents are available as a PDF, while others are available in HTML.

Some documents are more finished than others, and we use tags to distinguish between them:

  • summary, when we identified a user need and drafted a short description of what a document will do and consist of
  • draft, when we drafted a document and need feedback to learn if it meets user needs
  • published, when we published a document and it's ready to be used
